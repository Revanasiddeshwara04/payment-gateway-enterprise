# Enterprise Payment Gateway System

A production-grade payment gateway built using:

- Java 17
- Spring Boot 3
- Spring Cloud
- Microservices
- Eureka
- Config Server
- API Gateway
- JWT Security
- MySQL
- Kafka
- Docker
- Kubernetes
- AWS

## Services

- Config Server
- Service Registry
- API Gateway
- Auth Service
- Merchant Service
- Payment Service
- Transaction Service
- Refund Service
- Settlement Service
- Notification Service
- Fraud Detection Service
- Reporting Service
